## Run the API in local system

1. Fork the repository
2. Clone it 
```
git clone git clone https://github.com/<your_name>/Graphql-example
```
3. Install Dependencies
```
npm install
```
4. Start Server
```
npm start
```
5. Visit
```
localhost:3000/graphql
```
